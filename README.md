# michael-folio

An open-source portfolio template for developers to showcase their projects, coding skills, and personal story.

## Quick Start

```bash
# Create a new project using the michael-folio template
bun create astro@latest -- --template michaelkachiwala/michael-folio

# Follow the interactive prompts, then navigate to the project directory
cd <your-project-name>

# Start the local development server
bun dev
```

The development server will be available at `http://localhost:4321` by default.

## Deployment

The project can be built as a static site and deployed to any static hosting platform. For specific instructions on various platforms, refer to the [Astro Deployment Guide](https://docs.astro.build/en/guides/deploy/).
